# SFU
**SFU** - (***S**imple **F**iles **U**ploader*) загрузчик файлов на сервер 
# Описание :
Простой загрузчик файлов на сервер поддерживающий технологию **darg and drop** и сделанный по средствам **асинхроных запросов**
